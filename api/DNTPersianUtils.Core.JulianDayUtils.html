Remarks

This method is the exact inverse of PersianDayToJdn(int, int, int) and is based on the astronomical Persian calendar using a 2820-year leap cycle, as described in Calendrical Calculations by Reingold and Dershowitz.

The calculation is performed using integer arithmetic only and supports astronomical year numbering (including year 0).

PersianDayToJdn(int, int, int)

Converts a Persian (Solar Hijri) calendar date to its corresponding Julian Day Number (JDN).

public static long PersianDayToJdn(int year, int month, int day)

Parameters

year int

The Persian calendar year (astronomical year numbering; year 0 is supported).

month int

The Persian calendar month (1–12).

day int

The day of the month (1–31, depending on the month).

Returns

long

The Julian Day Number (JDN) corresponding to the specified Persian date.

Remarks

This implementation is based on the astronomical Persian calendar using a 2820-year leap cycle, as described in Calendrical Calculations by Reingold and Dershowitz.

The algorithm is purely integer-based, does not rely on floating-point arithmetic, and is valid for a wide range of years, including negative (astronomical) year numbering.

The Persian calendar has 682 leap years in every 2820-year cycle, resulting in an average year length of approximately 365.242198 days.
